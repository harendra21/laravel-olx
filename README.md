<h3>Olx</h3>

<h4>Technology Used </h4>
<ul>
    <li>Laravel 6.2</li>
    <li>MySql DB</li>
    <li>Jquery</li>
    <li>Bootstrap 4</li>
    <li>CSS</li>
    <li>HTML</li>
</ul>

<h4>Features </h4>
<ul>
    <li>Login/Registration</li>
    <li>Add, View and Delete product</li>
    <li>Real time comment using javascript(Jquery) short polling</li>
    <li>Multiple product image upload</li>
</ul>
